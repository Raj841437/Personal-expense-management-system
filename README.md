# Personal Expense Management System

A GUI-based desktop application developed using Python to help users record, manage, and analyze their daily expenses efficiently.

## Project Overview

The Personal Expense Management System is designed to provide a simple and user-friendly way to track personal expenses digitally. It replaces manual record keeping with a computerized system that ensures accuracy, security, and easy analysis of financial data.

This application allows users to:
- Add daily expenses
- View and manage expense records
- Edit and delete entries
- Analyze expenses category-wise using charts
- Securely log in using username and password

## Technologies Used

- **Python** – Core programming language  
- **Tkinter** – Graphical User Interface (GUI)  
- **SQLite** – Database for storing expense data  
- **Matplotlib** – Data visualization (charts)  
- **VS Code** – Development environment  

## Features

- User authentication (login system)  
- Add, edit, and delete expense records  
- Automatic total expense calculation  
- Category-wise expense analysis  
- Graphical visualization using bar charts  
- Local database storage using SQLite
